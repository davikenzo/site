Circuitos Digitais: Entrada, Processamento e Saída

Nesta aula, vamos explorar os fundamentos dos circuitos digitais, incluindo as portas lógicas, circuitos elétricos e a linguagem de programação C++.

Portas Lógicas

Porta AND (E)

Produz uma saída lógica HIGH apenas se todas as entradas forem HIGH.

Porta OR (OU)

Produz uma saída lógica HIGH se ao menos uma entrada for HIGH.

Porta NOT (NÃO)

Inverte a saída lógica de uma entrada.

Tabelas Verdade

As tabelas verdade são usadas para representar as relações entre as entradas e saídas de uma porta lógica.

Circuitos Elétricos

Circuitos elétricos são compostos por componentes elétricos, como chaves, fontes e resistores. Eles são usados para controlar o fluxo de corrente elétrica.

Portas Lógicas Integradas (ICs)

IC 7408

Contém portas AND, OR e NOT.

IC 7432

Contém portas AND e OR.

IC 7404

Contém portas NOT.

Linguagem C++ (C#)

A linguagem C++ é uma linguagem de programação popular usada para desenvolver aplicativos e sistemas. Ela é usada para programar microcontroladores, como o Arduino.

Exemplo de Código

Parte 1 - Declaração de Variáveis e Constantes

const int LED = 13;

Parte 2 - Configurações

void setup(){
  pinMode(LED, OUTPUT);
}

Parte 3 - Repetições

void loop(){
  digitalWrite (LED, HIGH);
  delay(1000); // 1 segundo
  
  digitalWrite (LED, LOW);
  delay(1000); // 1 segundo
}

Este código configura o pino 13 do Arduino como saída e, em seguida, alternadamente aciona e desaciona o LED com um intervalo de 1 segundo entre cada ação.

Conclusão

Nesta aula, exploramos os fundamentos dos circuitos digitais, incluindo as portas lógicas, circuitos elétricos e a linguagem de programação C++. Além disso, criamos um circuito simples com um LED, resistor e Arduino, utilizando a linguagem C++.
